Backend de Loja Virtual (Spring Boot + MySQL)

Aplicação Java baseada em Spring Boot cujo objetivo é expor uma API REST completa para gerenciar um sistema de loja virtual. Ele encapsula toda a lógica de negócio e persistência relacionada a usuários, produtos e pedidos, oferecendo:

  • Autenticação e Autorização com JWT, permitindo papéis (ROLE_USER e ROLE_ADMIN). Documentação automática de endpoints via Swagger/OpenAPI, usando interfaces definidas como contratos.
  • Tratamento unificado de erros por meio de um handler global, retornando respostas padronizadas.
  • Paginação e ordenação de listas de recursos (produtos e pedidos), para suportar grandes volumes de dados. Conversão entre entidades JPA e DTOs, garantindo que apenas as informações necessárias sejam expostas à camada externa.
  • Validações de entrada com Bean Validation (anotações como @NotNull, @Size, etc.) e exemplos/documentações via Swagger.

Frontend de Loja Virtual em Angular 19

Aplicação em Angular (v19+) que consome a API REST do ecommerce-backend. Em linhas gerais, ele deve permitir que o usuário:

  • Se registre e faça login (autenticação via JWT).
  • Navegue pela lista de produtos com paginação e filtros básicos.
  • Veja detalhes de cada produto.
  • Adicione produtos a um carrinho de compras.
  • Finalize o pedido (checkout), informando endereço e confirmando pagamento.
  • Usuários com papel ADMIN podem criar/editar/deletar produtos a partir de um painel administrativo.
  • Veja a lista de pedidos já realizados (para usuário comum) ou todos os pedidos (para ADMIN).
