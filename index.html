Projetos

Loja Virtual Full Stack (Angular 19 + Spring Boot + MySQL)

Projeto completo de uma aplicação full stack desenvolvida com Angular 19 no frontend, Spring Boot no backend e MySQL como banco de dados relacional. A solução visa simular um sistema de e-commerce moderno, robusto e seguro, com funcionalidades essenciais para o gerenciamento de usuários, produtos e pedidos.

Backend (Spring Boot)

  • API RESTful estruturada com Spring Boot, utilizando arquitetura em camadas (Controller, Service, Repository) e mapeamento com JPA para persistência no banco de dados MySQL.
  • Autenticação e Autorização baseada em JWT, com controle de acesso por perfis (ROLE_USER e ROLE_ADMIN).
  • Endpoints documentados automaticamente com Swagger/OpenAPI, com uso de interfaces como contratos da API.
  • Tratamento global de exceções via @ControllerAdvice, retornando mensagens padronizadas e amigáveis ao cliente.
  • Paginação e ordenação com Pageable, para facilitar navegação em grandes conjuntos de dados.
  • Conversão entre entidades e DTOs por meio de mapeadores dedicados, garantindo segurança e desacoplamento.
  • Validações robustas com Bean Validation (@NotNull, @Email, @Size, etc.), incluindo mensagens customizadas e integração com Swagger.

Frontend (Angular 19)

  • Interface SPA (Single Page Application) desenvolvida com Angular 19 e SCSS, proporcionando experiência fluida ao usuário.
  • Tela de login com autenticação via JWT, persistência de sessão e controle de acesso por perfil.
  • Listagem de produtos e pedidos com filtros, paginação e ordenação.
  • Formulários reativos com validações em tempo real, utilizando ReactiveForms.
  • Consumo de APIs RESTful usando HttpClient e serviços injetáveis.
  • Componentes reutilizáveis e estilizados conforme o handoff de UI.

Banco de Dados (MySQL)

  • Modelo relacional com tabelas normalizadas: users, products, orders e order_items.
  • Relacionamentos com integridade referencial (ex: um pedido contém vários itens).
  • Script de criação automática com Hibernate, pronto para migrações com Flyway ou Liquibase.
